Detailed Description

template<typename Element>
class EmojicodeCompiler::IDScoper< Element >

IDScoper is only a thin layer over a vector. It simply and strictly works like a table and relies on that SemanticScoper correctly issues VariableIDs, which are the indices by which the vector is accessed. Variables can therefore be retrieved in constant time. No real scoping mechanics are provided by this class, all scoping is implemented in SemanticScoper, which issues IDs so that rows of a IDScoper are reused once the corresponding variable dropped out of scope.
