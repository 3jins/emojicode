Detailed Description

template<typename TypeDef>
class EmojicodeCompiler::TypeBodyParser< TypeDef >

TypeBodyParser parses $type-body$s of $type-definition$s, which are represented by TypeDefinition. Some methods of this class are specialized for some types.
