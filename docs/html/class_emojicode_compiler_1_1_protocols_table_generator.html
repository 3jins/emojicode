Detailed Description

This class is responsible for declaring and generating individual protocol dispatch tables and the protocol table, which can then be stored into the box info or class info for value types and class types repectivley.

The protocol dispatch table map the protocol method VTI’s to the function the type which conforms to the protocol defined. One protocol table is created per type, which contains pointers to all protocols dispatch tables and allows dynamic casting to protocols.
