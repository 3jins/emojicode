Detailed Description

This class encapsulates the logic of deinitializing an object when initialization is aborted by raising an error.

An object whose initialization is aborted by an error cannot be deinitialized by the deinitializer as it might not be fully initialiized. Instead all instance variables that were indeed initialized must be released. This is what this class does.
