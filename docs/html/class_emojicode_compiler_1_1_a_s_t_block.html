void EmojicodeCompiler::ASTBlock::appendNodeBeforeReturn ( std::unique_ptr< ASTStatement >  node)

Inserts a statement before the return statement in this block.

Precondition
Calling this method is only valid if the last statement that is compiled is a return statement. stopsAtReturn() returns true in this case.

Determines whether the last statement of the block that is compiled (i.e. that is not dead code) is a return or raise statement.
