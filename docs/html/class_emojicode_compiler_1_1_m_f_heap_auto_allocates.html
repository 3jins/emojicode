Detailed Description

A class that either calls FunctionCodeGenerator::alloc or FunctionCodeGenerator::stackAlloc depending on whether MFHeapAllocates::allocateOnStack() was called, should inherit from this class, which encapsulates this logic and provides allocate() for allocation.
