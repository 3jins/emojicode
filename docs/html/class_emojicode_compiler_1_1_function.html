◆ vtiForUse()

int EmojicodeCompiler::Function::vtiForUse ( )

Returns the VTI for this function. If the function is yet to be assigned a VTI, a VTI is obtained from the VTI provider. If the function was not marked used, it will be. This function is a shortcut to calling assignVti and setUsed and then getting the VTI.

Warning
This method must only be called if the function will be needed at run-time and should be assigned a VTI.
